## 1.打开命令提示窗口

> `win` +` r`   输入`cmd` + `回车`

## 1.2常用命令

| 操作             | 说明                                       |
| ---------------- | ------------------------------------------ |
| e:               | 盘符切换到e盘（后面必须有：号）            |
| dir              | 查看当前路径下的内容（所有文件）           |
| cd 目录          | 进入单极目录                               |
| cd..             | 回退到上一级目录                           |
| cd 目录1\目录2.. | 进入多级目录                               |
| cd\              | 回退到盘符目录（就是回退到某个盘的最跟部） |
| cls              | 清屏                                       |
| exit             | 推出命令提示窗口                           |

## 1.3 dos命令创建文件夹：

> `md` 文件夹name

## 1.4 name补齐命令

某个文件夹下的文件首字母+`tab` 、这样可快速找到你想要的文件。
